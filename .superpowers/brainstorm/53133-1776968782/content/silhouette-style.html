Signal Silhouette — Visual Language

How the abstract signal profile renders alongside the body avatars

A — Aura Field

Body-shaped silhouette with colored aura layers. Each signal type (Spotify, Oracle, psychometrics) is a different color ring radiating outward. Overlap = where two people's auras bleed into each other. Organic, spiritual feel.

B — Constellation Map

Body reduced to points in space connected by lines — a human constellation. Each node's color and size encodes a signal dimension. Signal-specific glyphs orbit nearby. Overlap = shared stars, connected constellations. Celestial, data-forward.

C — Topographic Contour

Body shape rendered as elevation contour lines — like a topographic map of a person. Each contour ring is a signal layer, deeper = more aligned. When two people's contours are shown together, shared depth zones light up. Scientific, intimate, geological.
