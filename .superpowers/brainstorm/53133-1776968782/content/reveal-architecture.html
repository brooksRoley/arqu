Match Reveal — Surface Architecture

How the reveal unfolds as a sequence of moments

A

Scroll Narrative

Single page, vertical scroll. Each section fades in as you scroll: silhouette → desire overlap → signal layers (Spotify, Oracle, psychometrics) → "connect" CTA. Cinematic, slow, one match at a time. Think: a dossier you read top to bottom.

Flow
Signal Silhouettes ↓ Desire Overlap ↓ Spotify / Strava / Oracle ↓ Psychometric Alignment ↓ Message / Connect
B

Phased Reveal (Timed Sequence)

Automated animation — the match "materializes" over 15-20 seconds. Silhouettes emerge from noise, converge, overlap zones illuminate, signal layers bloom outward. User watches, then gets controls. Think: an unveiling ceremony.

Flow
Noise / Void → Silhouettes Emerge → Overlap Zones Glow → Signal Layers Bloom → Interactive Explore
C

Hybrid — Animated Intro + Scrollable Detail

Opens with a 5-8 second animated convergence (the emotional hit), then transitions into a scrollable detail view where each signal layer can be explored. First visit gets the animation; revisits from messages skip to the detail view.

First Visit
Animation → Silhouettes Converge → Scroll Detail View
Revisit (from messages)
Scroll Detail View (animation skipped)
